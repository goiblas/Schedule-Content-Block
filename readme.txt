== Installation & Setup ==

Note: This plugin requires WordPress 5

1. Upload 'Schedule-content-block' to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
4. Add Schedule Content Block to the content
5. Includes everything you want to program your visualization inside
